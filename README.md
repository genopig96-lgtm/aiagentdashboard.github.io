# Daisy Selfservice Report

Website báo cáo Selfservice tương tác.

## Cấu trúc

- `index.html`: trang báo cáo chính.
- `.nojekyll`: yêu cầu GitHub Pages phục vụ file tĩnh trực tiếp.

## Cập nhật báo cáo

1. Xuất lại báo cáo Selfservice từ dashboard.
2. Đổi tên file mới thành `index.html`.
3. Thay file cũ trong repository.
4. Commit thay đổi; GitHub Pages sẽ tự triển khai lại.
